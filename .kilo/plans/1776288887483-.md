# 项目审查计划

## 审查概述
对 Vancouver PWID 驱逐与暴力数据可视化项目进行全面代码审查，找出潜在问题。

---

## 发现的问题

### 1. 数据不一致 (严重)

**文件:** `Resources/Table 1 (Cleaned).csv:20`

```
Experienced violence*,223,245,111
```

- Males: 245 + Females: 111 = 356
- 而 Total 显示为 223
- **问题:** 总数与分类之和不相符，差值为 133

---

### 2. 无障碍访问问题

| 位置 | 问题 | 严重程度 |
|------|------|----------|
| 多个 SVG 图标 | 缺少 `aria-label` 或 `title` 属性 | 中 |
| 链接按钮 | 缺少键盘焦点样式 `:focus` | 中 |
| 图片 | Vancouver skyline 缺少 `loading="lazy"` | 低 |
| 页面 | 缺少 "skip to main content" 链接 | 低 |

---

### 3. 安全问题

| 位置 | 问题 | 建议 |
|------|------|------|
| `index.html:7-8` | CDN 脚本无 `integrity` hash | 添加 Subresource Integrity |
| `<head>` | 缺少 Content Security Policy | 添加 CSP meta 标签 |
| `index.html:112` | DOI 链接使用 `http://` | 改为 `https://` |
| `index.html:228-263` | Tableau iframe 无 `sandbox` 属性 | 添加限制性 sandbox |

---

### 4. SEO 问题

| 缺失项 | 位置 |
|--------|------|
| `<meta name="description">` | `<head>` |
| `<link rel="canonical">` | `<head>` |
| Open Graph 标签 | `<head>` |
| Twitter Card 标签 | `<head>` |

---

### 5. 性能问题

| 位置 | 问题 |
|------|------|
| `index.html:127-130` | 图片无 `loading="lazy"` |
| `index.html:12` | Google Fonts 阻塞渲染 |
| `index.html:170,208` | 静态图片可优化 |

---

### 6. 代码质量问题

| 位置 | 问题 |
|------|------|
| `index.html:14-17` | 混用中英文注释 |
| `index.html:248-263` | 内联脚本可外部化 |
| `index.html:159` | `!border-b-0` 破坏样式一致性 |

---

### 7. HTML 验证问题

| 位置 | 问题 |
|------|------|
| `index.html:103` | 期刊名斜体标签 `<i>` 应用 `<em>` 替代 |
| `index.html:134` | `target="_blank"` 缺少 `rel="noopener"` (已有) |

---

## 修复优先级

1. **高优先级:** 数据不一致问题
2. **中优先级:** 安全增强 (CSP, SRI)
3. **中优先级:** 无障碍修复
4. **低优先级:** SEO 元数据
5. **低优先级:** 代码清理

---

## 修复计划

1. 修正 Table 1 CSV 中的数据合计
2. 为 CDN 资源添加 integrity 哈希
3. 添加 CSP meta 标签
4. 修复 DOI 链接协议
5. 添加缺失的 meta 标签
6. 为图片添加 lazy loading
7. 清理注释并统一编码风格
